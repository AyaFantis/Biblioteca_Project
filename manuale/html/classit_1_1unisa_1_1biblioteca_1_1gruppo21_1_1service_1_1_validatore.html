◆ validaEmail()

boolean it.unisa.biblioteca.gruppo21.service.Validatore.validaEmail ( String email)
static

Verifica se una stringa rappresenta un indirizzo email istituzionale valido.

Precondition
L'input email può essere una stringa qualsiasi o null.
Postcondition
Il metodo restituisce true se e solo se:
  1. L'input non è null.
  2. L'input rispetta il pattern regex (dominio "@studenti.unisa.it").
Il metodo restituisce false in tutti gli altri casi.
Parameters
emailLa stringa da validare.
Returns
Esito della validazione.

boolean it.unisa.biblioteca.gruppo21.service.Validatore.validaMatricola ( String matricola)
static

Verifica se una stringa rappresenta una matricola valida.

Precondition
L'input matricola può essere una stringa qualsiasi o null.
Postcondition
Il metodo restituisce true se e solo se:
  1. L'input non è null.
  2. L'input è composto esattamente da 10 caratteri numerici [0-9].
Il metodo restituisce false in tutti gli altri casi
