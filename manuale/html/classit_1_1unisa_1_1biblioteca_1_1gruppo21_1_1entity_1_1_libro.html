Detailed Description

Questa classe è un'entità che incapsula le informazioni fondamentali di un libro, inclusa la gestione della quantità totale di copie possedute dalla biblioteca.

Definisce l'ordinamento naturale degli oggetti Libro. L'ordinamento viene effettuati alfabetica per Titolo. In caso di titolo identici, viene utilizzato l'Autore come criteria secondario.

◆ equals()

boolean it.unisa.biblioteca.gruppo21.entity.Libro.equals ( Object o)

Verifica l'ugualglianza logica tra due libri. L'identità di un libro è determinata univocamente dal suo codice ISBN. Titolo, autore e altri campi vengono ignorati nel confronto poichè l'ISBN identifica univocamente l'edizione.

  • Parameters
    oL'oggetto con cui confrontare l'istanza corrente.
    Returns
    true se l'oggetto passato è un Libro e ha lo stesso ISBN, falsa altrimenti.
