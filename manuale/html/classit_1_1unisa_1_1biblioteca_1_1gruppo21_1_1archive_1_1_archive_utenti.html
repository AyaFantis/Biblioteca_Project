Detailed Description

Questa classe specializza ArchiveAstratto per gestire la persistenza degli oggetti Utente su file di testo, ed implementa la logica di mapping CSV specifica per i dati anagrafici.

Utente it.unisa.biblioteca.gruppo21.archive.ArchiveUtenti.cerca ( String id)

Cerca un utente nell'archivio tramite la sua matricola. Implementazione della ricerca per Matricola.

Precondition
Il parametro id (Matricola) non deve essere null.
Postcondition
Restituisce l'oggetto Utente se esiste nella cache un utente con quella matricola.
Restituisce null se nessun utente corrisponde alla matricola data.
Parameters
idLa matricola dell'utente da cercare.
Returns
L'oggetto Utente se presente, altrimenti null.

Ricostruisce un oggetto Utente da una riga di testo CSV.
