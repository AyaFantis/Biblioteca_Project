Detailed Description

Questa classe implementa le regole di dominio relative ai prestiti, coordinando le interazioni tra Utenti, Libri e l'archivio storico dei Prestiti. Verifica la disponibilità delle copie e i limiti imposti agli utenti.

String it.unisa.biblioteca.gruppo21.service.ServicePrestiti.nuovoPrestito ( String matricola,
String codiceISBN )

Registra un nuovo prestito.

Precondition
Deve esistere un utente registrato con la matricola fornita.
Deve esistere un libro in catalogo con l'ISBN fornito.
Il libro deve avere almeno una copia disponibile (Copie > 0).
L'utente non deve avere già 3 libri in prestito.
Postcondition
Viene creato un nuovo oggetto Prestito con stato ATTIVO.
Il numero di copie disponibili del libro viene decrementato di 1.
Il prestito viene aggiunto alla lista dei prestiti attivi dell'utente.
Parameters
matricolaLa matricola dell'utente.
codiceISBNL'ISBN del libro.
Returns
Messaggio di esito.
